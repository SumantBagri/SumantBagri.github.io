Diffusion Models on Edge

Benchmarked and optimized latent diffusion model inference across heterogeneous edge and desktop hardware using PyTorch, TorchScript JIT, ONNX Runtime, and TensorRT. Achieved 4.7× latency reduction on RTX 3070 via TensorRT (1.17s → 0.25s per inference at 20 DDIM steps). Deployed and profiled on NVIDIA Jetson Nano (aarch64), observing ~87s baseline inference latency — 74× slower than RTX 3070 due to Maxwell GPU constraints.

Key Technical Highlights
  • Implemented full inference pipelines for two diffusion model families (LDM on CelebA-HQ 256 and Stable Diffusion v1.4) in pure PyTorch without relying on Hugging Face diffusers runtime.
  • Built custom PyTorch modules for UNet2D, UNet2DConditionModel, VQ-VAE/KL-VAE decoders, CLIP text encoder with BPE tokenizer, and three schedulers (DDIM, Euler Ancestral, DPM-Solver Multistep).
  • Evaluated four hardware-level optimization strategies: FP16 half-precision, TorchScript JIT tracing, ONNX export with ORT graph optimization, and TensorRT engine compilation.
  • Built a structured profiling harness recording denoising latency, decoding latency, GPU memory delta, and power draw across 10 iterations with warmup.
  • Produced a comprehensive dataset of 90+ benchmark CSV files across all device/optimization/metric combinations.

Synthetic Brain Tumor MRI Generation

Designed and implemented AGGrGAN, a novel multi-architecture GAN ensemble combining DCGAN, WGAN, WGAN-GP, and U-Net GAN to synthesize brain tumor MRI scans across four BraTS 2020 modalities. Integrated VGG-19 neural style transfer as a post-processing stage, improving perceptual image quality by ~8 dB PSNR. Validated synthetic utility via a downstream ResNet classifier: augmenting training data raised classification accuracy from 89.8% to 95.0%.

Key Technical Highlights
  • Implemented all four GAN architectures from scratch in PyTorch with custom U-Net discriminator, orthogonal/normal weight initialization, and per-epoch PSNR/SSIM logging.
  • GAN aggregation uses Sobel edge detection, Gaussian filtering, and pixel-level averaging across generator outputs.
  • Applied EDSR ×4 super-resolution (via OpenCV dnn_superres) to upscale 64×64 GAN outputs, recovering structural detail lost at constrained training resolution.
  • Conducted a 13-variant ablation study; best result: DCGAN + Style Transfer — PSNR = 29.64 dB, SSIM = 0.87 on T1-weighted MRI.
  • Built a custom NIfTI data pipeline using NiBabel to load BraTS 2020 volumetric scans and extract middle axial slices across four modalities.

Comparison of Sampling-Based Path Planners

Implemented and benchmarked three asymptotically-optimal sampling-based path planning algorithms from scratch: Fast Marching Trees (FMT*), Batch Informed Trees (BIT*), and Neural RRT* (NRRT*). Evaluated across 7 obstacle map types on four metrics: solution path cost, execution time, success rate, and collision check count — each as a function of sample count and batch size.

Key Technical Highlights
  • FMT* planner with KD-tree-accelerated ball-radius neighbourhood queries and analytically-derived connection radius r_n based on the Lebesgue measure of free space.
  • BIT* with full prune-and-resample logic, dual priority queues, and an admissible ellipsoidal informed set bounding the sample space.
  • Designed and trained NRRTCNN — a custom deep encoder-decoder network on a dual-branch ResNet backbone with Atrous Spatial Pyramid Pooling (ASPP) — producing per-pixel sampling probability heatmaps to bias RRT* sampling.
  • Generated CNN training labels by running A* over 200 random start-goal pairs across 8 obstacle categories; trained with BCE loss and ReduceLROnPlateau over 50 epochs.
  • Produced publication-quality comparison plots with 95% confidence interval bands using Seaborn and Matplotlib.

Autonomous Mazerunner Omnibot — "Veronica"

Designed and built a fully autonomous mechatronic robot capable of self-localization, maze navigation, and pick-and-place operations within a physical 10×6 cell maze, completing the full mission in a single autonomous run. Implemented hardcoded sequence-based localization in Arduino C++ and a probabilistic Bayes-filter localization system in MATLAB over a 7×15×4 discrete state space (420 states).

Key Technical Highlights
  • Probabilistic Bayes-filter localization (MATLAB) with Gaussian convolution motion model and sensor-matched measurement updates (p_hit=0.6, p_miss=0.2), converging when max(p) > 0.8.
  • BFS maze solver for real-time path planning from the robot's most probable grid position to target zones, accounting for obstacle-blocked cells from a precomputed navigability mask.
  • Engineered a three-tier PID control system (B1/B2/B3) for simultaneous obstacle proximity correction, rotational alignment, and wall-following distance maintenance.
  • Built a custom Arduino library (_veronica_) encapsulating sensor multiplexing, PID computation, and motor actuation — interfacing 8 HC-SR04 ultrasonic sensors via a 16:1 CD4067 MUX.
  • Designed and 3D-printed custom chassis components in SolidWorks across multiple revisions (revB–revF), exported as STL for FDM fabrication.
  • Implemented a 2-DOF servo-actuated gripper (pitch + grip axes) with HC-SR04-based short-range block detection trigger (≤3 cm) for pick-and-place operations.
